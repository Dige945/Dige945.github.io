# 卷积的几个性质

## 平移不变�?就好比刚刚说的，每一个Fliter就像是一个滑动窗口，这样在要识别的图上进行特定stride的移动的卷积计算时，Filter里面的值是不变的，因为这样你才能够对全图进行同一个特征的识别�?
而这个Fliter，也就是这个矩阵里面的值，对应了神经网络学习中weight权重的给定，我们也是不断通过深度学习，逐渐确定$\vec{w}$(在卷积神经网络中也就是filter)的值，以达到对图片识别的目�?
### 平移不变性的数学表示
下面是平移不变性的数学表示�?
在之前全连接层的时候，我们输入图片时把图片（例�? * 3）拉成一个一维向量（1 * 9）。现在我们要还原图片里面各个像素的位置信�?
将输入和输出变形为矩阵（宽度w和高度h�?这样层与层向前计算的过程中，需要学习的权重$w$需要变形为一个四维张�?$W_{4D}$)

画了个简图，应该很好理解

![img](/images/posts/example8.png)

数学符号表示�?$$h_{i,j}=\sum_{k,l}w_{i,j,k,l}x_{k,l} \\\\ =\sum_{a,b}v_{i,j,a,b}x_{ {i+a},{j+b} } $$

其中$h$是卷积层的输出（注意不是高了），V是W的重新索引（只是方便我们引出卷积的平移不变性）
$$ v_{i,j,a,b}=w_{i,j,i+a,j+b}$$

显然x的平移会导致h的平�?
$$h_{i,j}=\sum_{a,b}v_{i,j,a,b}x_{ {i+a},{j+b} } $$

我们的目标是v不应该依赖于（i，j），即不会随便坐标的变换而改�?
故解决方法是�?$$ v_{i,j,a,b} =v_{a,b}$$

$$h_{i,j}=\sum_{a,b}v_{a,b}x_{i+a,j+b}$$

这就�?维卷积（交叉相关�?
## 局部�?
当我们进行图像识别的时候，每一个filter在运算的时候，并不会考虑Receptive Field(感受�?之外的信息，仅仅针对于局部的图像

所以，当评�?h_{i,j}$时，我们不应该用远离$x_{i,j}$的参�?
解决方案：当$|a|,|b|>\Delta$时，使得$v_{a,b}=0$
$$h_{i,j}=\sum^\Delta_{a=-\Delta}\sum^\Delta_{b=-\Delta}v_{a,b}x_{i+a,j+b}$$

#卷积的代码实�?```
import torch
from torch import nn
from d2l import torch as d2l

def corr2d(X,K):
    """计算二维互相关运�?""
    ###X是输入矩阵，K是卷积核矩阵
    h,w = K.shape
    #输出矩阵Y的高度是输入矩阵的高度减卷积核的高度+1，宽度是输入矩阵的宽度减卷积核的宽度+1
    Y = torch.zeros((X.shape[0] - h + 1, X.shape[1] - w + 1))
    for i in range(Y.shape[0]):
        for j in range(Y.shape[1]):
            Y[i,j] = (X[i:i + h, j:j + w] * K).sum()
    return Y

```

##卷积�?class Conv2D(nn.Module):
    def __init__(self, kernel_size):

# 由X到Y的卷积核的学�?
```
conv2d = nn.Conv2d(1, 1, kernel_size=(1, 2), bias=False)

X = X.reshape((1, 1, 6, 8))
Y = Y.reshape((1, 1, 6, 7))
lr = 0.03
for i in range(10):
    Y_hat = conv2d(X)
    ##均方误差
    l = (Y_hat - Y) ** 2
    conv2d.zero_grad()
    l.sum().backward()
    # 迭代卷积�?    #梯度下降公式
    conv2d.weight.data[:] -= lr * conv2d.weight.grad
    #每两次迭代输出loss
    if (i + 1) % 2 == 0:
        print(f'epoch {i + 1}, loss {l.sum():.3f}')

print(conv2d.weight.data.reshape((1, 2)))
```
